    encode_query(list)

    Takes a map and turns it into proper query values.

    Example

    card_data = %{ card: %{

    number: 424242424242,
exp_year: 2014

    } }

    Stripe.URI.encode_query(card) # card[number]=424242424242&card[exp_year]=2014
